## 🚀 Validação Local

### Importante: Usar Servidor HTTP

Para validar localmente, **não abra o arquivo diretamente** (file://). O site usa PWA/Manifest que não funciona corretamente via file:// por política CORS.

Use um servidor HTTP local:

```bash
# Opção 1: npx serve (recomendado)
npx serve .

# Opção 2: Python
python -m http.server 8080

# Opção 3: Node.js
npx http-server -p 8080
```

Acesse: http://localhost:8080
